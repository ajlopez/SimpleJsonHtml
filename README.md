# SimpleJsonHtml

SimpleJsonHtml generates HTML from a JSON template and a JSON model.

## Installation

Via npm on Node:

```
npm install simplejsonhtml
```


## Usage

Reference in your program:

```js
var sjh = require('simplejsonhtml');
```
